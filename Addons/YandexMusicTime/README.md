# Аддон для отслеживания времени, проведенного в Яндекс Музыке
## Возможности
- ⏱️ Автоматическое отслеживание времени прослушивания
- 🎨 Настраиваемый внешний вид цифр
- 📍 Выбор позиции отображения на экране (включая центральное положение)
- 🎨 Динамический цвет времени, адаптирующийся к цвету заголовка страницы
- 🎨 Возможность использования статического цвета
- 💾 Сохранение статистики между сеансами
- 🔄 Возможность сброса накопленного времени
- 👁️ Минималистичный дизайн без фона
- 🎵 Опциональная иконка перед временем
## Настройки
### Настройки отображения
- **Показывать время** - включение/выключение отображения времени
- **Позиция времени** - выбор расположения (вверху по центру, верхний левый угол, нижний правый угол, нижний левый угол)
- **Статический цвет** - переключатель между динамическим цветом (из заголовка страницы) и статическим цветом
- **Выбор цвета** - настройка статического цвета текста (работает только при включенном статическом цвете)
- **Сбросить статистику** - очистка накопленного времени
### Дополнительные настройки
- **Размер шрифта** - регулировка размера от 12 до 32 пикселей
- **Показывать секунды** - отображение секунд
- **Показывать иконку** - отображение иконки 🎵 перед временем
